# PiPi 精靈圖片放這裡

把你手上的寵物圖片（PNG，去背最佳）改成以下檔名，放進這個 `public/pets/` 資料夾，
Demo 就會自動用真圖顯示；沒放的話會自動 fallback 成 emoji，不會壞掉。

| 檔名 | 對應外型 | 你提供的圖 |
|---|---|---|
| `pipi-default.png` | 素顏 PiPi（新手預設） | 橘色、無裝扮那隻 |
| `pipi-suit.png` | 西裝 PiPi（投資理財最高） | 灰色西裝那隻 |
| `pipi-chef.png` | 主廚 PiPi（烘焙料理） | 咖啡色廚師 + 杯子蛋糕那隻 |
| `pipi-scholar.png` | 學者 PiPi（語言學習） | 戴眼鏡 + 抱書那隻 |
| `pipi-fitness.png` | 健身 PiPi（健康健身） | （可用西裝那隻或另做） |
| `pipi-life.png` | 品味 PiPi（生活品味） | （可另做） |

> 建議尺寸 512×512 以上、正方形、透明背景。
> 檔名要完全一致（小寫、含 `.png`）。
